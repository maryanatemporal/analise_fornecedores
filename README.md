Análise de Fornecedores

Este projeto tem como objetivo analisar dados de fornecedores, realizando limpeza, tratamento de inconsistências e extração de insights para apoiar a tomada de decisão.

Contexto
Este projeto foi inspirado em um cenário real da área de Suprimentos (Compras), com foco na avaliação de fornecedores com base em preço, qualidade e tempo de entrega.

 Tecnologias utilizadas
- Python
- Pandas
- Numpy

 Etapas da análise
- Leitura e exploração dos dados
- Identificação de problemas
- Limpeza e tratamento de dados
- Análise exploratória
- Geração de insights

 Principais Insights

- Fornecedores com maior preço tendem a apresentar melhor avaliação.
- Fornecedores mais baratos podem impactar negativamente na qualidade e no tempo de entrega.
- O Frigorífico Sul apresenta melhor custo-benefício.
- A escolha do fornecedor ideal depende do objetivo do negócio.

 Conclusão

A análise demonstra a importância do tratamento de dados e da análise exploratória para apoiar decisões estratégicas, considerando múltiplos fatores além do preço.
